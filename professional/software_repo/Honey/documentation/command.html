Command line reference

(Back to index)

Event script command line syntax is EventScript {script file} [options]

Possible options are:

  • --bequiet : Do not display information on the console. This option is useful the pipe out the processing results.
  • --filearg {file} : Read the file and interpret its content as the command line arguments.
  • --input {file} : Define the input dataset for the processing.
  • --output key:value : Define non-signal variable value for the script execution.
