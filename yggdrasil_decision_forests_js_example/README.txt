Yggdrasil Decision Forests Javascript Example
=============================================

This directory contains a live running example of the Yggdrasill Decision Forests Javascript+WebAssembly port. This example is available [here]().

This example is assambled from:

- The [YDF JS+WASM example](https://github.com/google/yggdrasil-decision-forests/tree/main/yggdrasil_decision_forests/port/javascript/example).
- The [Precompiled release of Javascript + WebAssembly RC1 for YDF v0.2.5](https://github.com/google/yggdrasil-decision-forests/releases/tag/js_0.2.5_rc1).
